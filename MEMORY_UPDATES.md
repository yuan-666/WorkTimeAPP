# 记忆修改登记

本文件用于给后续 AI 交接项目记忆。每次修改本项目时，都必须同时更新：

- `PROJECT.md`：更新当前项目事实、结构、规范或交接状态。
- `MEMORY_UPDATES.md`：新增一条本次修改登记。

登记格式建议：

```text
## YYYY-MM-DD HH:mm CST - 简短标题

- 触发原因：
- 修改文件：
- 行为变化：
- 验证结果：
- 风险/注意：
- 后续建议：
```
