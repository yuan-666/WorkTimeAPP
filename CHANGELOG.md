# Changelog

## v0.3.2 - 2026-05-13

### Added

- 手机月历页四宫格：税前/总工时（点击切换总加班）/目标/放假（点击切换上班天数）。
- 月历支持压缩到本周（收起/展开），折叠跨月自动回退到第一周。
- 月份切换（‹ 今天 ›）独立一行，右上角明暗切换按钮。
- 暗色模式全面修复：新增 `--accent-ring`、`--warn-ring`、`--holiday-ring`、`--muted-ring` 等主题变量。

### Changed

- 工资规则卡只出现在设置页；休息提醒只出现在月历页；总工资只出现在月历和报表页。
- 手机版日历格子、metric 卡片、topbar 整体缩小，适配一屏展示。

### Fixed

- 修复暗色模式下 `.notice` 白底白字不可读问题。
- 修复 `--text` 变量未定义导致部分文字颜色失效。
- 修复全部硬编码 rgba 边框在暗色模式下不可见的问题。
- 补全 `.chart-legend`、`.report-empty-state`、`.settings-desktop-save`、`.cloud-session-row`、`.cloud-task-grid`、`.mobile-login-state` 等 7 个缺失样式。

### Improved

- 精简引导页、设置 helper、登记按钮等文案，去掉 AI 风格措辞。
- 三 agent 并行审查修复了 `.mobile-cal-topbar` 对齐、按钮高度覆盖、month-nav 样式继承、dead code 等 7 个问题。

## v0.3.1 - 2026-05-13

### Added

- 云备份登录成功后发放 7 天用户会话；上传本机和恢复云端可直接使用登录状态，不再每次重复输入密码。
- 新增后端测试覆盖用户会话 token、同账号无密码同步、跨账号 token 拒绝和云端冲突强制覆盖。

### Fixed

- 修复手机端从设置二级页切到其他主页面后，再回到设置仍停留在旧二级页的问题；主导航进入设置会回到设置主菜单。
- 修复设置页下拉框在部分移动端 WebView 中闪现后消失的问题：设置详情容器不再裁剪原生下拉层，并显式保留系统原生选择框样式。
- 修复云端已有数据时上传失败后没有直觉路径的问题；现在会提示“是否用本机数据覆盖云端”，确认后才强制覆盖。
- 修复侧边栏和手机底部云备份登录态只能看不能点的问题；点击后会直接打开设置里的云备份登录区。

### Improved

- 设置页“云备份”拆成登录区和备份区：未登录只显示账号/密码/登录/创建，已登录显示上传本机、恢复云端和退出登录。
- 桌面设置页新增固定保存区，避免用户改完设置后找不到保存按钮。
- 云同步文案改为面向普通用户，明确“上传本机到云端”和“从云端恢复到本机”的数据方向。
- 桌面侧边栏品牌名右侧改为版本号胶囊链接，底部原版本位置改为当前云备份账号状态。
- 移动端底部去掉版本号和友情链接，只保留云备份登录状态与版权说明两行。
- 顶部规则状态卡改为纯静态展示：直接显示正班时薪、工作日/休息日/节假日加班时薪和下一次休息日，不再混入每日登记表单。
- 顶部规则状态卡右侧只保留“规则”按钮，不再额外显示“当前规则”文字。
- 报表年度总结增加颜色说明：绿色代表税后收入，黄色代表总工时；当前年份未来月份不再显示 0 元工资条。
- 版本线调整为补丁号到 9 后进入下一小版本：原 `v0.2.10` 归并为 `v0.3.0`，当前版本为 `v0.3.1`。

## v0.3.0 - 2026-05-13

### Fixed

- 修复手机端云备份登录读取不到账号密码的问题：云同步按钮现在优先读取当前二级设置页所属表单。
- 修复移动端点击日期、关闭登记抽屉、切换设置二级页时的闪现感：整页入场动画只在主页面切换时触发，登记抽屉改为两阶段打开/关闭。
- 修复设置二级页取消勾选后无法可靠保存的问题：`autoFillWorkday`、`autoAdjustment.enabled`、工作日勾选等字段按表单实际包含情况重置。
- 修复移动端切换薪资方式后字段不刷新的问题：切换时先保留当前表单草稿，再刷新对应薪资字段。
- 修复批量处理切换“批量添加/批量删除”会丢日期范围和输入的问题，并补上 Enter 提交处理。
- 修复管理员接口收到坏密文时返回 `Failed to decode base64` 等底层错误的问题。

### Improved

- 手机端设置页的底部导航保留四个主入口，设置详情页使用页内返回，减少“进设置后找不到 Tab”的困惑。
- 管理员登录配置更宽容：`admin_name` 会自动去除前后空格，`admin_passwd` 兼容标准 `passwordSalt/passwordHash`、简写 `salt/hash`、明文字符串和常见 KV 包装。
- `RSA_key` 和 `admin_passwd` 读取时会兼容误复制的三反引号代码围栏，降低部署配置踩坑概率。
- 新增后端登录测试，覆盖 RSA 公钥获取、管理员加密登录和坏密文安全失败。

## v0.2.9 - 2026-05-13

- 修复薪资模式切换在移动端触发全页重渲染：仅桌面端调用 `render()`。

## v0.2.8 - 2026-05-13

### Fixed

- 修复批量添加点击后无反应的问题：放弃 form submit 方案，改用 click 事件处理器直接读取表单数据，避免表单在渲染过程中被断开。
- 修复批量添加切换"添加方式"后条件字段（加班小时、班次选择）需要重新渲染导致表单丢失的问题：所有字段始终显示，不再做条件渲染。
- 修复侧边栏版本信息文字溢出重叠：简化为版本号+日志链接两列布局，增加溢出保护。
- 修复调休上班日在日历和状态栏重复显示标签的问题：`dayMeta` 直接使用 `holiday.name`。
- 修复周合计行工资显示为 ¥0.00 的问题：对无记录但开启 autoFillWorkday 的工作日，计算默认 8 小时的工资。
- 修复班次管理保存后丢失加班班次的问题：所有班次模板（包括休息日加班）都保留在表单中，仅视觉隐藏。
- 修复云备份操作缺少具体错误提示的问题：解析服务端 HTTP 状态码和错误消息，区分账号不存在、密码错误、账号已停用、数据冲突等场景。

### Added

- 新增 `autoFillWorkday` 设置（默认开启）：无记录的工作日自动算作每日正班小时数，`calculateMonthlyPayroll` 在计算时自动合并虚拟条目；日历对自动补登的天数显示"默认"标记和虚线边框；小时计算模式下自动跳过。
- 新增 `getAutoFilledEntries()` 计算函数，扫描当月工作日并生成虚拟正班条目。
- 设置页新增"上班方式"选择器：固定白班、固定夜班、轮班（白班+夜班）、自定义，选择后自动配置对应班次模板（含加班班次）。
- 设置页新增"默认班次"选择器，决定工作日自动使用哪个班次模板。
- 设置页新增"关于"分组，显示应用图标、版本号、更新日志/GitHub/博客链接和版权信息。
- 移动端设置页改为二级页面：点击分组进入独立详情页，顶部有返回箭头按钮。
- 移动端主页面切换时增加 slide-in 过渡动画。
- 明暗模式切换按钮改为 SVG 太阳/月亮/显示器图标。
- 新增 5 个测试用例覆盖 `getAutoFilledEntries` 和 `autoFillWorkday` 开关，测试总数从 36 增加到 41。

### Improved

- 侧边栏版本信息区域简化：移除"次更新"计数，只保留版本号和"查看日志"链接。
- 移动端设置页从展开式改为菜单列表+二级详情页模式，更符合手机 App 交互习惯。
- 移动端底部导航在设置页显示返回按钮+页面标题，替代底部 Tab 栏。
