# Changelog

## v0.1.1 - 2026-05-11

### Fixed

- 修复快速加班可被重复点击叠加到不合理工时的问题；同一天只维护一条快速加班记录。
- 修复默认班次、复制上次会不断新增同日主记录的问题；现在会替换当天主记录，避免误操作堆叠。
- 修复手动工时可填写 200 小时等异常值的问题，新增单条、当天、加班和休息时间校验。
- 修复工时记录中“总小时”和“正班+加班”不一致时可能导致工资计算失真的问题。
- 修复 PWA 更新后可能继续读取旧缓存的问题，Service Worker 改为在线优先、离线回退。

### Improved

- 每日记录增加当天合计、正班/加班、当天上限和预计工资概览。
- 表单增加内联错误提示，异常输入会显示“不可保存”。
- 自动识别并合并历史同日同类重复记录，顶部提示仍需检查的异常日期。
- 设置和班次输入增加合理上限，降低误填成本。
- 测试从 14 项增加到 17 项，覆盖异常工时、时间范围和当天合计上限。

## v0.1.0 - 2026-05-11

### Added

- 离线优先 PWA：支持桌面和移动浏览器安装使用。
- 月历工作台：按日查看工时、工资、加班和补扣状态。
- 工时记录：支持时间记录、工时记录、跨天班次和休息扣除。
- 薪资模式：支持正班+加班、底薪+加班、综合工时制、小时计算。
- 自动化能力：支持班次模板、默认班次、复制上次、只记加班、按工作日批量生成。
- 工资推导：支持由底薪和月标准小时自动推算正班时薪、底薪时薪和加班基数。
- 补贴扣款：支持补贴、扣款和日补扣自动记录。
- 个税估算：支持居民工资薪金累计预扣法默认税率表。
- 报表导出：支持 CSV、Excel、分享摘要和 JSON 备份导入导出。
- 测试覆盖：核心计算逻辑包含 14 项 Node 测试。

### Notes

- 数据默认保存在本机浏览器 `localStorage`，建议每月发薪后导出备份。
- 本应用提供工资与个税估算，不构成劳动法、税务或财务意见。
