# Changelog

## v0.1.5 - 2026-05-12

### Added

- 工作日默认套用工作班次；白班、夜班等工作班次时间不一致时，日记录会提示先选择今日班次。
- 请假从备注入口独立出来，支持年假、事假、病假、调休、是否扣工资、计薪倍数、请假小时和扣款金额。
- 新增放假提醒，可选择按每周休息日、上六休一、上十四休一或自定义周期，并根据上一次休息日计算下一次建议休息日。
- 个税扣除项改为“固定金额 / 工资比例”二选一，社保公积金和其他扣除分别配置。
- 日记录表单新增实时工时预览，显示当天总工时、正班、加班和预计工资。

### Improved

- 日历周视图改为周一开始。
- 记录页移除示例按钮，保留批量添加和醒目的批量删除操作。
- 移除劳动法合规提醒和倍率下限兜底，倍率按用户设置计算。
- 滚动页面时会结束正在编辑的时间/数字输入，减少移动端滑动时误改数字。
- 测试从 23 项增加到 24 项，覆盖周一开周、休息周期、请假计薪和自定义倍率。

## v0.1.4 - 2026-05-12

### Added

- 新增“正常上班 / 有加班 / 休息 / 请假备注”四种登记入口，用户按真实状态选择即可。
- 新增批量处理面板，支持按日期范围、工作日规则和班次模板批量添加记录。
- 新增批量删除能力，可删除批量生成、加班、全部工时、补扣或全部记录；删除前显示影响并要求确认。
- 支持休息和请假/备注的 0 工时记录，不再强迫用户填无意义小时。

### Improved

- 每日登记默认只显示状态判断、班次、上班/下班/休息和主保存按钮，日期类型、记录方式、目标、备注默认收进“更多设置”。
- 日历单日面板弱化分散快捷按钮，把复制昨天和批量处理放到面板头部。
- 批量删除后可撤销，批量添加默认不覆盖已有记录，降低误操作风险。
- 测试从 22 项增加到 23 项，覆盖 0 工时休息/请假标记。

## v0.1.3 - 2026-05-11

### Added

- 内置国务院办公厅 2026 年部分节假日安排，自动识别法定节假日、放假休息日和调休上班日。
- 日历格增加 `法定`、`休`、`班` 标记，选中日期后展示当天适用工资倍率。
- 登记表单增加日期类型分段选择和班次预计工资预览，节假日默认按对应倍率计算。

### Improved

- 重做每日快捷登记区域：默认班次一键保存更醒目，复制上次、快捷加班、补贴扣款的动作更清楚。
- 批量生成工作日记录会跳过节假日和周末，并自动包含国务院调休上班日。
- 班次设置新增日期类型，保存设置后“休息日加班”等班次不会丢失休息日口径。
- 测试从 20 项增加到 22 项，覆盖节假日覆盖工作周、调休上班和法定节假日工资倍率。

## v0.1.2 - 2026-05-11

### Fixed

- 修复“小时计算”模式未对加班小时应用法定倍率的问题；小时工现在也会按正班时薪拆分正班工资和加班工资。
- 修复用户将工作日、休息日、法定节假日倍率误填低于法定值时可能少算工资的问题；计算层会按 1.5x / 2x / 3x 下限兜底。
- 修正综合工时制默认口径：月标准工时按 20.67×8=165.36 小时，超出审批周期标准工时按不低于 1.5x，法定节假日按不低于 3x。
- 修复只填写总小时的工时记录无法正确拆分正班/加班的问题。

### Improved

- 薪资设置页增加法定口径说明：月薪折算小时工资按 21.75 天计薪，即月工资 ÷ 174。
- 日记录列表展示正班工资、加班工资和当前适用倍率，便于复核每笔工资。
- 报表页增加法定口径校验，提示月加班 36 小时、工作日每日加班 3 小时、周工时 44 小时等风险。
- 测试从 17 项增加到 20 项，覆盖小时工加班倍率、倍率下限和合规提醒。

## v0.1.1 - 2026-05-11

### Fixed

- 修复快速加班可被重复点击叠加到不合理工时的问题；同一天只维护一条快速加班记录。
- 修复默认班次、复制上次会不断新增同日主记录的问题；现在会替换当天主记录，避免误操作堆叠。
- 修复手动工时可填写 200 小时等异常值的问题，新增单条、当天、加班和休息时间校验。
- 修复工时记录中“总小时”和“正班+加班”不一致时可能导致工资计算失真的问题。
- 修复 PWA 更新后可能继续读取旧缓存的问题，Service Worker 改为在线优先、离线回退。

### Improved

- 每日记录增加当天合计、正班/加班、当天上限和预计工资概览。
- 表单增加内联错误提示，异常输入会显示“不可保存”。
- 自动识别并合并历史同日同类重复记录，顶部提示仍需检查的异常日期。
- 设置和班次输入增加合理上限，降低误填成本。
- 测试从 14 项增加到 17 项，覆盖异常工时、时间范围和当天合计上限。

## v0.1.0 - 2026-05-11

### Added

- 离线优先 PWA：支持桌面和移动浏览器安装使用。
- 月历工作台：按日查看工时、工资、加班和补扣状态。
- 工时记录：支持时间记录、工时记录、跨天班次和休息扣除。
- 薪资模式：支持正班+加班、底薪+加班、综合工时制、小时计算。
- 自动化能力：支持班次模板、默认班次、复制上次、只记加班、按工作日批量生成。
- 工资推导：支持由底薪和月标准小时自动推算正班时薪、底薪时薪和加班基数。
- 补贴扣款：支持补贴、扣款和日补扣自动记录。
- 个税估算：支持居民工资薪金累计预扣法默认税率表。
- 报表导出：支持 CSV、Excel、分享摘要和 JSON 备份导入导出。
- 测试覆盖：核心计算逻辑包含 14 项 Node 测试。

### Notes

- 数据默认保存在本机浏览器 `localStorage`，建议每月发薪后导出备份。
- 本应用提供工资与个税估算，不构成劳动法、税务或财务意见。
