# 明薪工时

明薪工时是一个离线优先的跨平台工时、加班、薪资和个税记录 PWA。Windows、macOS、iOS、Android、鸿蒙都可以通过现代浏览器运行，移动端可添加到主屏幕。

当前版本：`v0.1.0`

## 功能

- 日历视图记录每日工时、加班、目标和备注
- 支持时间记录与工时记录
- 支持正班+加班、底薪+加班、综合工时制、小时计算
- 支持班次模板、一键套用默认班次、复制上次、只记加班
- 支持按工作日批量生成本月记录
- 支持底薪和月标准小时自动推算正班时薪、底薪时薪和加班基数
- 支持补贴、扣款、月度薪资拆分、年度统计
- 支持日补扣自动记录
- 支持居民工资薪金累计预扣法个税估算
- 支持 CSV、Excel、分享摘要、本地 JSON 备份与导入
- PWA 离线缓存，数据默认保存在本机浏览器 `localStorage`

## 运行

```bash
npm test
npm run serve
```

然后打开 `http://localhost:4173`。

## 使用流程

1. 在 `设置` 中选择薪资方式，填写底薪、月标准小时、倍率、个税扣除和默认班次。
2. 回到 `月历`，点击日期后可套用默认班次、复制上次、只记加班，或手动填写上下班时间。
3. 在 `记录` 中可按本月工作日批量生成记录，也可录入补贴和扣款。
4. 在 `报表` 中查看年度趋势、本月工资拆分，并导出 CSV 或 Excel。
5. 定期使用 `备份` 导出本地 JSON，避免浏览器清理数据造成丢失。

## 计算口径

加班倍率默认使用工作日 1.5 倍、休息日 2 倍、法定节假日 3 倍，可在设置中调整。

个税默认使用居民个人工资薪金累计预扣法：累计收入减累计减除费用、专项扣除、专项附加扣除等后，套用 3% 至 45% 的累计预扣率表。默认累计减除费用按 5000 元/月计算。参考国家税务总局政策解读：<https://www.chinatax.gov.cn/chinatax/n810341/n810760/c3959585/content.html>

本应用提供工资与个税估算，不构成劳动法、税务或财务意见。具体发薪、社保、公积金、补休和税务申报请以合同、当地政策与单位规则为准。
