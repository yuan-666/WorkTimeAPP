# Release Notes - v0.1.6

发布日期：2026-05-12

## 概览

`v0.1.6` 继续把“每天记一笔”做轻：少填、少猜、少误点。重点是让上班、加班、请假和批量处理都更像实际工作流，而不是一张复杂表格。

## 重点改进

- 每日登记更快：新增准点下班、加 1/2/3 小时快捷调整。
- 请假更顺手：新增全天、半天、2 小时、自定义快捷时长，并实时显示请假工资。
- 保存前更有数：保存栏显示本条工资、本月税后预估和目标差额。
- 防重复更强：同一天已有主记录时，再保存会替换主记录，避免重复叠加工时。
- 批量处理更清楚：批量添加和批量删除拆成两个模式，删除前只看删除影响和确认项。
- 记录列表更像人话：请假、休息按真实状态展示，不再伪装成 0 小时工时。
- 设置页更干净：个税扣除、社保公积金、放假周期只显示当前选项需要的字段。
- 撤销更从容：删除、覆盖、导入等高风险提示停留更久。
