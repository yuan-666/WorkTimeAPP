# Release Notes - v0.1.2

发布日期：2026-05-11

## 概览

`v0.1.2` 是一次工资计算口径修正版本，重点修复小时工加班未按法定倍率计算的问题，并把加班工资、综合工时、小时工资折算和合规提醒统一到一套计算规则里。

## 重点改进

- 小时工加班修正：小时计算模式现在会拆分正班工资和加班工资，工作日加班按不低于 1.5 倍计算。
- 法定倍率兜底：工作日 1.5x、休息日未补休 2x、法定节假日 3x。设置里误填更低倍率时，系统仍按法定下限计算。
- 月薪折算统一：默认小时工资 = 月工资 ÷ 21.75 ÷ 8，即月工资 ÷ 174。
- 综合工时修正：默认月标准工时为 165.36 小时，超出标准工时按不低于 1.5x，法定节假日按不低于 3x。
- 工资明细更好查：每日记录展示正班工资、加班工资和适用倍率。
- 合规提醒更明确：报表页提示月加班 36 小时、工作日每日加班 3 小时、周工时 44 小时等风险。
