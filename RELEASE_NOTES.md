# Release Notes - v0.3.4

发布日期：2026-05-14

## 概览

`v0.3.4` 是一次针对用户体验（UX/UI）的细节打磨和本地开发体验升级版本。重点修复了部分组件在极端场景（如桌面端暗色模式）下的显示问题，并大幅改善了本地联调云备份的流程。

## UX / UI 细节打磨

- **Toast 提示框修复**：解决了暗色模式下提示框 `.notice` 在桌面端被异常推高（`bottom: 130px`）的 Bug，现在各种分辨率下均能完美贴合。
- **更柔和的交互反馈**：将表单输入框获取焦点（Focus）时的边缘发光由生硬的纯色高亮改为“分层渐变阴影”，触感更加原生高级。
- **「今天」的优雅标记**：去除了日历格子中代表「今天」的僵硬边框，改为蓝色字号加粗的文本标记，视觉更清爽，更贴合 Apple HIG 理念。
- **设计规范对齐**：更新了项目内的 `DESIGN.md` 规范文档，全面废弃老版青绿风格说明，确保文档与现行实际落地的 iOS Blue 风格保持一致。

## 本地开发升级

- **Node.js 代理服务**：移除了只能托管静态文件的本地 Python 服务器，新增 `scripts/serve.mjs`。
- **本地 Mock API**：通过本地拦截 API 请求并读写 `.local-kv.json`，完美模拟了阿里云 ESA 环境。现在运行 `npm run serve` 后即可直接在 `127.0.0.1:4173` 测试全部的云端账号注册、登录与同步流程，再也不会报 404 错误。

## 技术

- 测试 45 项全部通过，构建通过。
