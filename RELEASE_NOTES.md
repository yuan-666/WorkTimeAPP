# Release Notes - v0.2.10

发布日期：2026-05-13

## 概览

`v0.2.10` 是一次移动端体验和登录链路修复版本。重点解决手机端点击闪现、设置二级页保存不准、批量处理输入丢失，以及云备份/管理员登录配置容易失败的问题。

## 修复内容

- 手机端云备份登录现在会读取当前设置二级页表单，不再只找桌面端设置表单。
- 日历登记抽屉改为两阶段打开/关闭，遮罩和抽屉有稳定过渡，减少“点击一下闪现”的观感。
- 移动端整页入场动画只在主页面切换时触发，日期选择、批量预览、设置详情切换不再反复整页动画。
- 设置详情页保存会正确处理取消勾选，`autoFillWorkday`、自动补扣、工作日勾选不再保留旧值。
- 移动端切换薪资方式会保留当前草稿并刷新对应字段。
- 批量处理会保留日期范围、添加方式、加班小时、删除范围和确认状态；表单按 Enter 也会执行当前模式。
- 管理员接口对坏密文返回友好错误，不再暴露 base64/decrypt 底层异常。

## 部署与配置改进

- `admin_name` 自动去掉前后空格。
- `admin_passwd` 兼容标准 PBKDF2 字段、`salt/hash` 简写、明文字符串和常见 KV 包装。
- `RSA_key`、`admin_passwd` 兼容误复制的三反引号代码围栏，但仍建议 KV 里只保存干净 JSON 或字符串。

## 测试

- 43 项测试全部通过。
- 新增后端测试覆盖 RSA 公钥获取、管理员加密登录和坏密文安全失败。
- 所有核心 JS 文件语法检查通过。
- `npm run build` 成功。
