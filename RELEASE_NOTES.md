# Release Notes - v0.2.9

发布日期：2026-05-13

## 概览

`v0.2.9` 是 `v0.2.8` 的 bug 修复版本，解决了班次管理无法切换、设置详情页保存丢失数据等问题。

## 修复内容

- 修复班次管理无法切换上班方式（workType change handler 未匹配详情页表单）。
- 修复切换上班方式后全页闪烁（改用局部 DOM 更新）。
- 修复设置详情页保存后丢失其他设置（只重置表单中实际包含的字段）。
- 修复设置详情页保存后不返回列表。
- 修复薪资模式切换在移动端触发全页重渲染。

## 测试

- 41 项测试全部通过。
- 所有 JS 文件语法检查通过。
- `npm run build` 成功。
