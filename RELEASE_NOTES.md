# Release Notes - v0.1.5

发布日期：2026-05-12

## 概览

`v0.1.5` 继续优化实际工厂、门店和轮班场景：工作日默认有班，用户主要确认白班/夜班、请假、休息和批量操作。系统不再用劳动法合规提醒打断记录流程，而是按用户设置的工资规则计算。

## 重点改进

- 工作日默认班次：自动套用工作班次；白班/夜班时间不一致时，日记录提示先选今天上哪个班。
- 请假登记：支持年假、事假、病假、调休、是否扣工资、计薪倍数、请假小时和扣款金额。
- 批量删除更明确：记录页保留批量添加/删除，移除示例按钮。
- 周一开周：日历按周一到周日展示。
- 放假提醒：支持双休、上六休一、上十四休一和自定义周期，可按上一次休息日反推下一次建议休息日。
- 个税设置更清楚：其他扣除、社保公积金都可选择固定金额或工资比例，按累计预扣法估算。
- 实时工时预览：填写上下班和休息时间时，立即显示当天工时、正班、加班和预计工资。
- 移动端输入保护：滑动页面会结束正在编辑的时间/数字输入，减少误改。
