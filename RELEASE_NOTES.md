# Release Notes - v0.3.5

发布日期：2026-05-14

## 概览

`v0.3.5` 是一次关键的后端（云同步）热修复版本。彻底解决了由于阿里云边缘安全加速（ESA）平台限制导致的数据云同步报错 500 问题。

## 关键修复：突破 Serverless KV 限制

- **背景**：在 `v0.3.4` 及之前版本中，如果尝试进行数据注册、上传等操作，用户经常会遭遇 500 Internal Server Error。根本原因是阿里云 ESA 平台在单次云函数（Edge Function）请求中，强制硬性限制最多只能调用 8 次 KV（Key-Value 数据库）的读取或写入。而我们应用复杂的加密防刷流程导致最高会产生 10 次请求。
- **内存级缓存加速**：在新版本中，我们将针对 IP 防刷限流（Rate Limit）的计数器、以及公钥解析所需的高频 `RSA_key` 证书读取操作，整体搬迁到了 Edge 节点的“短生命周期内存”中。同时，删减了不会被后台读取的冗余元数据（`userMetaKey`）写入。
- **结果**：原本一次云备份需要足足消耗 10 次数据库调用，现在成功被压降至平均只有 5~6 次，为服务器预留了充足的安全余量。目前线上环境同步恢复如丝般顺滑！
