# Release Notes - v0.1.0

发布日期：2026-05-11

## 概览

`v0.1.0` 是明薪工时的首个可用 PWA 版本，目标是让小时工、加班工、底薪加班和综合工时用户能用一个跨平台网页应用完成每日工时记录、薪资估算、补扣管理、年度统计和导出备份。

## 重点能力

- 每日记录更快：班次模板、套用默认班次、复制上次、只记加班。
- 工资设置更简单：底薪和月标准小时可自动推导时薪和加班基数。
- 月度管理更清晰：月历展示每日工时、加班和工资，记录页支持批量生成工作日。
- 发薪复核更方便：报表提供工资拆分，Excel 导出包含年度汇总和每日明细。
- 本地优先更稳妥：应用可离线缓存，支持 JSON 备份和导入。

- 已在本地浏览器验证默认班次保存、批量生成、设置页渲染和移动端视口。

## 已知限制

- 法定节假日暂未自动识别，需要手动选择日期类型。
- 综合工时制目前以月度目标小时估算，后续可扩展为审批周期统计。
- 数据保存在浏览器本地，跨设备同步需要后续引入云同步或加密备份。
