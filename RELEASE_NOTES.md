# Release Notes - v0.1.4

发布日期：2026-05-12

## 概览

`v0.1.4` 是一次人机交互改版，目标是把“填表记工时”改成“选择今天发生了什么”。用户默认只需要选状态、选班次、确认时间，系统负责日期类型、加班拆分、工资预估和节假日倍率。

## 重点改进

- 单日登记更自然：新增 `正常上班`、`有加班`、`休息`、`请假/备注` 四种入口。
- 表单更轻：默认只显示班次、上班、下班、休息和主保存按钮，复杂字段收进 `更多设置`。
- 休息和请假更合理：支持 0 工时记录，休息不再被迫填写无意义小时。
- 批量添加：可按日期范围、工作日/休息日规则和班次模板生成记录，默认不覆盖已有记录。
- 批量删除：可按范围删除批量生成、加班、全部工时、补扣或全部记录；删除前显示影响并要求确认，删除后可撤销。
