# Release Notes - v0.3.6

发布日期：2026-05-14

## 概览

`v0.3.6` 专注于提升日历核心交互的流畅度。我们为月份翻页操作添加了平滑的动画效果，让应用在视觉上更具连贯性和原生感。

## 日历翻页平滑动画

- **视觉连贯性**：消除了之前月份切换时日历内容“瞬间跳转”的生硬感。
- **智能方向识别**：
  - 点击“上个月”或向右轻扫时，内容从左侧滑入。
  - 点击“下个月”或向左轻扫时，内容从右侧滑入。
- **全平台适配**：该动画同时支持桌面端的按钮点击操作和移动端的手势轻扫切换。
