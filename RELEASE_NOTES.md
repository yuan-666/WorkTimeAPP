# Release Notes - v0.3.3

发布日期：2026-05-14

## 概览

`v0.3.2` 是一次 iOS 原生风格全面重构版本。从玻璃拟态彻底转向纯色 Apple HIG 设计，新增四宫格手机日历、滑动翻页、密码修改和 API 安全加固。

## 视觉重构

- **iOS 原生 HIG 风格**：`#F2F2F7` 极浅灰底 + `#FFFFFF` 纯白卡片 + 细微阴影，暗色 `#000000` 纯黑底 + `#1C1C1E` 深灰卡片。
- **微交互**：所有可点击元素按下时 `scale(0.96)` 物理收缩，弹簧缓动 `cubic-bezier(0.25, 1, 0.5, 1)`。
- **无玻璃拟态残留**：移除所有 `backdrop-filter`、`blur`、`saturate`、弥散光背景。

## 手机日历

- 四宫格：税前 / 距目标 / 总上班（X天 Xh）/ 总加班（Xh），休息倒计时独立长条。
- 月份切换「‹ 回到今天 ›」位于日历顶部，支持左右滑动翻页（≥60px）。
- Bottom Sheet 录入面板：32px 大圆角，弹簧动画滑出，物理拖拽把手。
- 日历格子缩小（`min-height` 48→40px），整月一屏展示。

## 修复

- 明暗切换按钮图标不可见（SVG `color` 继承修复）。
- 选中日期背景白色导致文字不可见（CSS 级联修复 + `!important` 胜利规则）。
- 云备份 ESA KV 调用次数超限（单次请求 `RSA_key` 缓存复用）。
- 桌面端月份切换被误删（恢复于 topbar 中）。

## 新增

- 密码修改：登录后在云备份区验证旧密码后修改，RSA-OAEP 加密传输。
- API 安全：IP 限流 10/min、请求体 ≤500KB、字段白名单。
- 左右滑动手势翻页（日历页专属）。

## 技术

- 版本 `v0.3.2`，Service Worker 缓存名 `worktimeapp-v25`。
- 测试 45 项全部通过。
