# Release Notes - v0.3.9

发布日期：2026-05-15

## 概览

`v0.3.9` 是一次倒班体验和发布稳定性版本。它让倒班规则参与云同步，增加“距本次班次结束”提示和工时班次时间复用，同时解决桌面窗口缩窄后的横向溢出。现在在窄桌面、平板和手机宽度下，报表数字、日期输入、工资设置和倒班规则都能稳定收缩，不会把页面撑出边界。

## 体验修复

- **窄桌面不溢出**：720-1120px 宽度提前切换为更稳的单列/两列布局，桌面窗口稍微缩窄时仍可正常使用。
- **报表数字更稳**：本月指标卡、年度总结条、薪资拆分和图表图例增加数字收缩、换行和省略保护。
- **日期填写不撑破**：记录筛选、批量处理、工资设置和倒班设置中的日期/时间/数字输入框都增加了宽度保护。
- **倒班设置更可靠**：白班、上夜班、下夜班、休班周期行在窄宽度下会提前换列，避免班次名称和时间选择器挤出容器。
- **切页更干净**：页面入场动画从横向位移改为轻微纵向位移，避免切换页面时短暂出现横向滚动条。
- **备份导入按钮修复**：隐藏的文件 input 不再撑宽设置页。

## 倒班日历

- 倒班功能仍然是可选功能：未开启时隐藏倒班导航、倒班页面和详细规则；开启后显示独立倒班页。
- 默认周期为白班、上夜班、下夜班、休班四段，班次名称可按用户自己的班组叫法修改。
- 新增周期开始时间，今日班次会按锚点日期和开始时间判断当前属于哪一天的班次。
- 倒班页底部详情改为“本次结束”和“距本次结束”，不再展示意义不大的轮次数字。
- 轮班规则可从工时班次模板复用上下班时间，白班、夜班不用重复录入。
- 倒班页会展示今日班次、本轮进度、下个休班日期和当月班次统计。

## 云同步

- 云备份会保存工时记录、工资设置、倒班规则和本地备份元数据。
- 后端会同时保留独立 `shiftCalendar` 快照和 `settings.shiftCalendar`，恢复旧备份或新备份都能拿到倒班规则。
- 管理员看板用户摘要可看到是否开启倒班和周期天数，但不会暴露用户工时明细。

## 技术

- 版本 `v0.3.9`，Service Worker 缓存名 `worktimeapp-v32`。
- 入口 HTML、模块导入和 Service Worker 注册参数均升级到 `v0.3.9`。
- 核心测试 50 项通过；本地浏览器在 756px 宽度扫描月历、报表、记录、设置和倒班页，未发现页面级横向溢出。
