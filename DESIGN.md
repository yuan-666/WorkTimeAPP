# 明薪记 UI 设计系统 v1.0

> 为「明薪记」PWA 打造的全新视觉设计系统，以清晰、高效、亲和为设计原则，让工时记录变得轻松自然。

---

## 设计原则

1. **一目了然** — 用户打开应用后，3 秒内知道今天该做什么、本月挣了多少
2. **少点一次** — 常用操作不超过 2 次点击，支持快捷预设和智能默认
3. **信息分层** — 核心数据突出显示，次要信息优雅收折，高级功能不打扰
4. **随时安心** — 离线可用、数据安全、操作可撤销，给用户掌控感
5. **跨屏一致** — 桌面端高效操作，移动端单手可达，视觉语言统一

## 色彩系统

本应用采用原生 iOS HIG (Human Interface Guidelines) 色彩体系，强调内容和交互状态，减少装饰性元素。

### 主色调 (Primary)

| Token | 色值 (Light) | 色值 (Dark) | 用途 |
|-------|------|------|------|
| `--bg` | `#F2F2F7` | `#111216` | 页面全局背景，深色模式避免纯黑 |
| `--surface` | `#FFFFFF` | `#202126` | 主要卡片、导航栏、底部抽屉 |
| `--surface-2` | `#F2F2F7` | `#2A2C33` | 次要背景、输入框、辅助按钮 |
| `--surface-hover`| `#E5E5EA` | `#373A43` | 卡片与元素的悬停/按下状态 |
| `--accent` | `#007AFF` | `#0A84FF` | **主品牌色**：主按钮、强调文字、选中标记 |
| `--accent-soft` | `rgba(0, 122, 255, 0.1)` | `rgba(10, 132, 255, 0.15)` | 轻量选中背景、提示条底色 |
| `--on-accent` | `#FFFFFF` | `#FFFFFF` | 主按钮上的文字色 |

### 功能色 (Semantic)

| Token | 色值 (Light) | 色值 (Dark) | 用途 |
|-------|------|------|------|

| `--warn` | `#FF3B30` | `#FF453A` | 危险、删除、错误提醒 |
| `--danger-soft`| `rgba(255, 59, 48, 0.1)` | `rgba(255, 69, 58, 0.15)` | 错误状态的柔和底色 |

### 中性色 (Neutral)

| Token | 色值 (Light) | 色值 (Dark) | 用途 |
|-------|------|------|------|
| `--ink` | `#000000` | `#FFFFFF` | **主文字色**、主要标题、重要数据 |

深色模式完全由 CSS 变量体系驱动 (`[data-theme="dark"]` 或 `@media (prefers-color-scheme: dark)`)。所有卡片在近黑背景上使用多层深灰 (`#202126` / `#2A2C33`) 承载内容，避免纯黑和低对比透明玻璃层。

## 组件规范

### 按钮 (Button)

**主按钮**
- 背景：`--primary-500`
- 文字：白色，14px，字重 700
- 内边距：10px 20px
- 圆角：`--radius-md`
- Hover：背景 `--primary-400`，translateY(-1px)，阴影 `--shadow-md`
- 按下：translateY(0) scale(0.98)
- 禁用：opacity 0.5，无交互效果

**次要按钮**
- 背景：白色
- 边框：1px solid `--gray-200`
- 文字：`--gray-900`，14px，字重 500
- Hover：背景 `--gray-50`，边框 `--gray-300`

**危险按钮**
- 背景：`--danger-50`
- 边框：1px solid `--danger-200`
- 文字：`--danger-500`，14px，字重 700

**图标按钮**
- 尺寸：40px × 40px
- 圆角：`--radius-md`
- 无背景，Hover 时背景 `--gray-100`

### 输入框 (Input)

- 高度：42px
- 背景：白色
- 边框：1px solid `--gray-200`
- 圆角：`--radius-md`
- 内边距：9px 12px
- Focus：边框 `--primary-500`，外发光 `0 0 0 3px rgba(23,107,91,0.12)`
- 错误状态：边框 `--danger-500`，背景 `--danger-50`
- 禁用：背景 `--gray-100`，文字 `--gray-400`

### 卡片 (Card)

- 背景：白色
- 边框：1px solid `--gray-200`
- 圆角：`--radius-lg`
- 内边距：`--space-4`（桌面端 16px，移动端 12px）
- 可选阴影：`--shadow-md`
- Hover（可选）：translateY(-2px)，阴影 `--shadow-lg`

### 日历格子 (Day Cell)

- 宽高比：1:0.86（桌面端），自适应（移动端）
- 最小高度：86px
- 圆角：`--radius-md`
- 边框：1px solid `--gray-200`
- 日期数字：14px，字重 800
- 工时/工资：12px，字重 600
- 选中状态：边框 `--primary-500`，内阴影 inset ring
- 有工时：渐变背景 `linear-gradient(180deg, white, --primary-50)`
- 节假日：边框 `--warning-500`，背景 `--warning-50`
- 休息日：背景 `--gray-50`
- 今日：日期数字使用 `--primary-500`
- 自动补登：虚线边框

### 数据指标卡片 (Metric Card)

- 背景：白色 + backdrop-blur
- 边框：1px solid `--gray-200`
- 圆角：`--radius-md`
- 内边距：16px
- 标签：13px，`--gray-400`
- 数值：24px，字重 700，`--gray-900`
- 4 列网格（桌面端），2 列（移动端）

### 底部抽屉 (Bottom Sheet)

- 移动端专用
- 圆角：`--radius-xl`（顶部）
- 最大高度：86vh
- 背景：白色
- 阴影：`--shadow-xl`（上方）
- 打开动画：translateY(100%) → 0，300ms ease-out
- 遮罩：rgba(0,0,0,0.3) + backdrop-blur(2px)
- 拖拽手柄：顶部居中，42px × 4px，圆角条

### 导航 (Navigation)

**桌面侧边栏**
- 宽度：260px
- 背景：半透明毛玻璃 `rgba(255,255,255,0.78)` + backdrop-blur(18px)
- 右边框：1px solid `--gray-200`
- 品牌区：图标 + 名称 + 版本胶囊
- 导航项：圆角 8px，padding 12px 14px
- 选中：背景 `--primary-100`，文字 `--gray-900`
- 未选中：文字 `--gray-400`

**深色玻璃**
- 背景：`rgba(32,33,38,0.9)`，保留平台玻璃和 blur，但不让底层内容穿透影响可读性
- 边框：`rgba(255,255,255,0.18)`，配合内高光形成原生 App 式层级
- 底部导航、移动抽屉、设置详情均使用同一玻璃表面；内部输入框和列表块使用 `--surface-2` 提高文字与控件对比度

**移动端底部导航**
- 固定底部，左右 12px 边距
- 背景：半透明毛玻璃
- 圆角：`--radius-lg`
- 阴影：`--shadow-lg`
- 4 个等宽项，图标 + 文字
- 选中：文字 `--primary-500`
- 安全区适配：`env(safe-area-inset-bottom)`

## 动效规范

### 入场动画

- 页面切换：opacity 0→1 + translateX(12px)→0，200ms ease
- 卡片出现：opacity 0→1 + translateY(8px)→0，250ms ease-out，stagger 50ms

### 交互反馈

- 按钮按下：translateY(1px) scale(0.98)，100ms
- 按钮 hover：translateY(-1px)，阴影增强，180ms
- 输入框 focus：边框颜色过渡 + 外发光，150ms
- 卡片 hover：translateY(-2px)，阴影增强，200ms

### 状态切换

- 底部抽屉打开：translateY(110%→0) + opacity，300ms `cubic-bezier(0.16, 1, 0.3, 1)`
- 底部抽屉关闭：translateY(0→110%) + opacity，220ms
- 遮罩淡入淡出：opacity，180ms
- 日历格子选中：边框颜色 + 内阴影，150ms

### 数据更新

- 数字变化：建议添加计数动画（可选）
- 列表增删：高度展开/收缩，200ms ease
- 通知出现：从右侧滑入，300ms；自动消失，opacity 淡出

## 无障碍设计

### 键盘导航

- 所有交互元素可通过 Tab 键访问
- 焦点顺序遵循视觉层级
- 焦点指示器：3px solid `--primary-500`，2px 偏移

### 屏幕阅读器

- 图标按钮带 `aria-label`
- 表单字段带关联 `<label>`
- 状态变化通过 `aria-live` 区域通知
- 日历格子标注日期和状态

### 色彩无障碍

- 所有文字与背景对比度 ≥ 4.5:1（WCAG AA）
- 不单独依赖颜色传达信息（配合图标/文字）
- 支持系统偏好 `prefers-reduced-motion`

### 触控适配

- 触控目标最小 44px × 44px
- 移动端按钮适当增大
- 防止误触的间距
