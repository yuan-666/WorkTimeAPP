Changelog

产品更新轨迹

这里按版本记录真正影响使用体验的变化：登记是否更顺手，工资是否更清楚，备份和同步是否更可靠。

01 v0.4.4

原生外壳与深色玻璃修复

2026-05-18
  • 修复 macOS 26 Electron 包启动崩溃，补齐 Hardened Runtime 下 Electron Framework 需要的 library validation 例外。
  • 深色模式玻璃层提高表面不透明度、边框和文字对比，底部导航、抽屉、设置详情和输入框更清楚。
  • Android 原生壳新增 WorkTimeNative 桥接和返回键处理，并加入 Kyant0/AndroidLiquidGlass 依赖基础。
  • iOS 原生壳改为自定义 Bridge ViewController，注入 Apple native shell 状态并衔接系统材质背景。
  • 新增 HarmonyOS ArkTS/ArkWeb 初始工程骨架，可在 DevEco Studio 中接手本地或线上 PWA 壳。
02 v0.4.3

App 化平台体验正式版

2026-05-18
  • 登记抽屉、设置详情、云备份改密和倒班详情统一支持系统返回与手势返回，先关闭子层，再回到主界面。
  • 平板和窄桌面进入 App 式布局，保留底部导航、登录状态页脚、移动月历和设置 drill-in。
  • 底部导航、抽屉和设置详情增强类 Liquid Glass 玻璃材质，Android/HarmonyOS 采用更圆润的视觉变量。
  • 新增平台识别、显示模式、指针类型和握持偏好标记，为后续原生壳桥接左右手识别预留接口。
  • 自分发脚本只收集当前版本桌面包，避免旧版 DMG/ZIP 混入新版发布目录。
03 v0.4.2

App 化返回手势与玻璃层

2026-05-17
  • 登记抽屉、设置详情、云备份改密表单和倒班详情统一进入 App 内返回栈，返回手势会先关闭当前子层。
  • 平板和窄桌面宽度也使用底部导航、移动月历和设置二级菜单，不再像网页表单被压窄。
  • 移动端倒班日期点击后打开详情抽屉，可查看本次结束、结束倒计时和周期信息。
  • 底部导航、抽屉、设置详情和云备份改密表单增强玻璃材质，Android/HarmonyOS 使用更圆润的类 Liquid Glass 视觉。
  • 新增平台识别和握持习惯设置，为未来鸿蒙/原生壳同步左右手识别结果预留安全桥接。
04 v0.4.1

明薪记品牌与自分发

2026-05-17
  • 产品展示名称改为“明薪记”，名字更短，也更明确地表达“工时记清楚、工资算明白”。
  • 更新 PWA、Android、iOS、Electron、后台和更新日志的展示名称，同时保留技术包名和云备份命名，避免影响升级数据。
  • 换用青绿色记录卡片与金色工资符号的新图标，并同步生成 Android、iOS、Electron 所需原生图标资源。
  • 新增自分发脚本，可生成 GitHub Release 或网盘分发目录，自动带安装说明、发布清单和 SHA-256 校验文件。
  • README 和发布说明补充 Android APK、桌面安装包、iOS PWA 优先等非应用商店安装策略。
05 v0.4.0

原生打包与发布安全加固

2026-05-17
  • 新增 Capacitor Android/iOS 工程，Android 目标 SDK 升级到 36，支持边到边显示、预测返回、主题图标和安全 WebView 配置。
  • 新增 Electron 桌面壳，提供 macOS/Windows/Linux 打包脚本，并在 macOS 使用系统材质窗口与安全导航限制。
  • 移动端底部导航加入图标、返回键能关闭登记抽屉和设置详情，原生壳会自动使用线上云同步接口。
  • 云备份接口加强来源校验、请求体大小限制、登录限流、会话版本失效、数据清洗和安全响应头。
  • 新增原生图标/启动图生成脚本，README 和发布说明补充 Android、iOS、macOS、Windows、Linux 的打包流程。

09 v0.3.5

突破 Serverless KV 限制

2026-05-14
  • 修复线上部署环境报错 500 的严重问题。
  • 采用 Edge 边缘节点内存直读替换冗余的 ESA KV 数据读取。
  • 将会话注册、云备份同步等操作单次 KV 调用次数从超标的 10 次压降至 5 次左右。
  • 突破了阿里云边缘安全加速（ESA）服务严格的「单次请求最多调用 8 次 KV API」的性能防线，云同步恢复丝滑流畅。
