Changelog

产品更新轨迹

这里按版本记录真正影响使用体验的变化：登记是否更顺手，工资是否更清楚，备份和同步是否更可靠。

01 v0.3.9

倒班云同步与窄桌面修复

2026-05-15
  • 倒班规则加入云备份，工时、设置和倒班周期可以一起跨设备同步。
  • 倒班页底部详情改为“本次结束”和“距本次结束”，正在上班时直接显示剩余时长。
  • 倒班规则可从工时班次模板复用上下班时间，减少重复填写白班/夜班时间。
  • 修复桌面窗口缩窄后报表数字、年度总结条、日期输入和设置表单横向溢出的问题。
  • 720-1120px 宽度提前进入更稳的单列/两列布局，平板和窄桌面不再把内容挤出边界。
02 v0.3.7

倒班日历

2026-05-15
  • 新增可开关的倒班日历页，开启后主导航显示“倒班”，关闭后隐藏倒班导航、页面和规则设置。
  • 支持按周期第 1 天日期自动推算白班、上夜班、下夜班、休班，班次名称可自行修改。
  • 倒班页展示今日班次、本轮进度、下个休班和当月班次统计。
  • 手机端倒班月历更紧凑，点击日期只查看班次，不会误弹出工时登记抽屉。
03 v0.3.6

日历翻页平滑动画

2026-05-14
  • 为日历月份切换引入了平滑的滑动过渡动画。
  • 无论是点击顶部导航按钮，还是在移动端左右轻扫，日历都会以线性优雅的方式进入视野。
  • 彻底解决了之前月份切换时内容瞬间跳转、视觉体验生硬的问题。
04 v0.3.5

突破 Serverless KV 限制

2026-05-14
  • 修复线上部署环境报错 500 的严重问题。
  • 采用 Edge 边缘节点内存直读替换冗余的 ESA KV 数据读取。
  • 将会话注册、云备份同步等操作单次 KV 调用次数从超标的 10 次压降至 5 次左右。
  • 突破了阿里云边缘安全加速（ESA）服务严格的「单次请求最多调用 8 次 KV API」的性能防线，云同步恢复丝滑流畅。
05 v0.3.4

UX/UI 细节打磨与本地 API 服务

2026-05-14
  • 修复暗色模式下桌面端提示框 `.notice` 悬浮过高的 Bug。
  • 柔化输入框 Focus 状态的分层渐变阴影。
  • 日历「今天」去除边框，改用蓝字加粗标记，界面更清爽。
  • 重写设计规范 `DESIGN.md`，从老版青绿风格对齐到当前的原生 iOS Blue 风格。
  • 用 Node.js 替换本地 Python 服务器，新增本地 Mock KV 和 API 转发，解决本地开发 404 问题。
06 v0.3.3

iOS 原生风格重构与拇指区交互

2026-05-14
  • iOS 原生 HIG 风格：极浅灰底 + 纯白卡片 + 细微阴影，暗色纯黑底。
  • 手机版四宫格（税前/目标/总上班/总加班），休息倒计时独立长条。
  • 月份切换「‹ 回到今天 ›」，左右滑动翻页。
  • 修复明暗切换图标不可见、选中日期空白（白底白字）。
  • 云备份新增密码修改、API 限流 10/min、请求体 ≤500KB。
  • 全局微交互 scale(0.96) 物理收缩、弹簧缓动。
07 v0.3.1

云同步与移动端设置修复

2026-05-13

2026-05-13
